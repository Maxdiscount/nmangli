# **App Name**: Mangli Admin Plus

## Core Features:

- Product Management: Add, edit, and disable products with details like name, price, unit, category and image. Utilizes local storage.
- Category Management: Add and disable categories to organize products.
- Admin Authentication: Secure the admin panel with password-based authentication.
- Product Image URL Validator: Uses an AI tool to confirm that product URLs correspond to valid images. It will flag non-working URLs or those that return error codes.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to evoke trust and reliability.
- Background color: Very light Indigo (#F0F2FA) for a clean, uncluttered feel.
- Accent color: Soft Purple (#9575CD) for interactive elements and highlights.
- Font pairing: 'Inter' (sans-serif) for body and headlines.
- Use consistent and clear icons from Lucide to represent actions and categories.
- Use clear card-based layouts to organize products and categories within the admin panel. Maintain consistency in padding and spacing.
